# Date-Time-Script
B7d Date & Time Script (English) :

This resource provides a clean and modern Date & Time display for FiveM, built specifically for QBCore servers. It is designed to show the current in-game date and time through a sleek and unobtrusive interface that fits naturally into any roleplay environment.

The Date & Time widget uses a custom UI designed by B7d Development, featuring a minimalist layout that stays fixed on the screen without interfering with gameplay. Its compact design ensures players can easily check the date and time while remaining fully immersed in roleplay.

The interface focuses on clarity and style, using carefully selected fonts, colors, and spacing to maintain readability while preserving a professional look. The resource is lightweight, optimized for performance, and safe to run on both small and large servers.

This script is easy to install and customize, allowing server owners to adjust styling, positioning, and behavior to match their server’s unique theme.

Key features include :

Real-time in-game date and time display

Clean and modern custom UI by B7d Development

Compact, fixed-position Date & Time widget

Lightweight and performance-friendly

Easy configuration and customization

Fully compatible with QBCore framework

Suitable for roleplay and custom UI servers

This Date & Time script enhances immersion by providing players with a simple and reliable way to track in-game time, while maintaining a polished and professional appearance.
